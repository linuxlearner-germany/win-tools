# 🪟 Windows Setup Toolkit

> ⚡ Automatisiere dein Windows-Setup in Minuten – sauber, schnell und reproduzierbar.

---

## 📦 Features

* 🧰 **Automatische Software-Installation** (winget)
* ⚙️ **Interaktive Tool-Auswahl**
* 🚀 **Schnelles Setup für neue Systeme**
* 🔄 **Einfach erweiterbar**
* 🧪 Ideal für **Developer, Power-User & IT**

---

## 🚀 Quick Start

```powershell
git clone https://github.com/DEIN-USERNAME/win-setup.git
cd win-setup/scripts
.\install.ps1
```

## ⚙️ Verwendung

Starte das Script:

```powershell
.\install.ps1
```

Dann kannst du:

* einzelne Tools auswählen → `1,3,5`
* alle installieren → `A`
* abbrechen → `Q`
